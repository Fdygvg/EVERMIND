QUESTION= Can users send data directly to databases
ANS=Not exactly — you can’t write action="mongodb" directly, because:
The action must be a URL or file path (something the browser can send a request to).
MongoDB is a database, not a web page or file.
Browsers can’t send data directly to a database — only to a server (like a PHP, Node.js, or Python file) that then talks to the database for you.
