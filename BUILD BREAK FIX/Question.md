# Question:
  How do you wirte a JSDoc Comment
## Answer:

```js
It’s structured documentation meant for humans and tools.

What type of comment is this?
/**
 * @desc    Get today's smart review questions
 * @route   GET /api/smart-review/today
 * @access  Private
 */


✅ JSDoc / Docblock comment

Key giveaway:

Starts with /**

Uses @tags

Each line starts with *

What it’s actually for (real purpose)

This comment is used to:

Document API routes

Generate API docs automatically

Help other devs understand intent instantly

Power tools like Swagger / OpenAPI

It does NOT affect runtime behavior.
It’s for documentation only.
```
